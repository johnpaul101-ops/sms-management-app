# 📱 SMS Management App

A production-ready, full-stack web application designed for managing, sending, and tracking SMS records and notification logs. This project is architected using the MERN stack with a clean separation of concerns between the Frontend client and the Backend API.

## 🚀 Key Features
- **User Authentication:** Secure user registration and login flows implemented via JSON Web Tokens (JWT) and encrypted passwords.
- **Data Management:** Complete CRUD (Create, Read, Update, Delete) capabilities for handling text logs, contacts, and message records.
- **Optimized Frontend UI:** A fast, single-page application dashboard built with React and Vite for near-instant rendering.
- **Secure Backend Architecture:** RESTful API design featuring production-grade request validation, error-handling middleware, and protected routing.

## 🛠️ Tech Stack

**Frontend:**
- React.js (Vite)
- Axios (HTTP client for asynchronous API communication)
- Tailwind CSS

**Backend & Database:**
- Node.js & Express.js (REST API Layer)
- MongoDB & Mongoose (Object Data Modeling)
- JSON Web Tokens (JWT) & Bcrypt (Authentication & Security)
